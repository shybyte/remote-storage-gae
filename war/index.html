Remote Storage for your Google Account alpha

This is a remotestorage server, that you can use for remoteStorage 0.7 web apps.

If your gmail address is for example username@gmail.com then you can now use username@remote-storage.appspot.com as remote storage address to login in remoteStorage web apps.

Please try one of these apps as example:
  • Unhosted Time Tracker

FAQ

Will my data be safe?
No, it won't yet. For now this service is meant as an option for developers who don't have their own remotestorage server or want to test their apps against multiple storage providers. Don't be suprised if your data disappear. In the near future this service will become a user friendly, effortless and reliable remote storage provider.
Where is the code?
Find it on github.
Where can I report bugs or propose new features?
If this service gives you any trouble or you have great ideas for improvement, please open an issue.
